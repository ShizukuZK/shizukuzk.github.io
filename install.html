How to install the modpack (NeoForge 1.21.1)

1. Requirements

  • Java 17+ (recommended)
  • Minecraft 1.21.1 compatible NeoForge
  • At least 4 GB RAM assigned to the client

2. Download the modpack

Click the "Open Modpack" link on the homepage or use the direct Google Drive link. After download, verify the SHA256 checksum (displayed on the download page).

3. Install NeoForge

  1. Open your launcher and add a new installation for Minecraft 1.21.1.
  2. Install NeoForge 1.21.1 and point the launcher to it.
  3. Create a fresh profile and assign at least 4 GB RAM.

4. Import modpack

If your launcher supports import, import the provided profile. Otherwise, copy the `mods` and `config` folders into your `.minecraft/mods` and `.minecraft/config` directories.

5. Common issues & fixes

  • Crash on launch: Check the launcher logs, remove any extra mods not in the pack, and ensure correct NeoForge version.
  • Missing dependencies: Re-download the modpack and confirm all ZIPs were extracted.
  • Black screen or texture errors: Update GPU drivers and increase JVM memory.
