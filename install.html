How to install the modpack (NeoForge 1.21.1)

1. Requirements

  • Java 17+ (recommended)
  • Minecraft 1.21.1 compatible NeoForge
  • At least 4 GB RAM assigned to the client

2. Download the modpack

Click the "Open Modpack" link on the homepage or use the direct Google Drive link. After download, verify the SHA256 checksum (displayed on the download page).

3. Install NeoForge

  1. Open your launcher and add a new installation for Minecraft 1.21.1.
  2. Install NeoForge 1.21.1 and point the launcher to it.
  3. Create a fresh profile and assign at least 4 GB RAM.

4. Import modpack

If your launcher supports import, import the provided profile. Otherwise, copy the mods and config folders into your .minecraft directories.

5. Common issues & fixes

  • Crash on launch: Check the launcher logs, remove any extra mods not in the pack, and ensure correct NeoForge version.
  • Missing dependencies: Re-download the modpack and confirm all ZIPs were extracted.
  • Black screen or texture errors: Update GPU drivers and increase JVM memory.

Как установить модпак (NeoForge 1.21.1)

1. Требования

  • Java 17+ (рекомендуется)
  • NeoForge для Minecraft 1.21.1
  • Минимум 4 ГБ ОЗУ для клиента

2. Скачай модпак

Нажми кнопку "Открыть модпак" на главной странице или используй прямую ссылку на Google Drive. После скачивания проверь контрольную сумму SHA256.

3. Установи NeoForge

  1. Открой лаунчер и создай новую установку для Minecraft 1.21.1.
  2. Установи NeoForge 1.21.1 и выбери его в настройках.
  3. Создай новый профиль и выдели минимум 4 ГБ ОЗУ.

4. Импортируй модпак

Если твой лаунчер поддерживает импорт, используй его. Иначе скопируй папки mods и config в твою папку .minecraft.

5. Частые проблемы и их решения

  • Краш при запуске: Проверь логи лаунчера, удали ненужные моды, убедись что выбран правильный NeoForge.
  • Отсутствуют зависимости: Скачай модпак заново и убедись что все ZIP-архивы распакованы.
  • Чёрный экран или ошибки текстур: Обнови драйверы видеокарты и увеличь JVM память.
