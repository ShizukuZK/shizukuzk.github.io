
Static one-page site for GitHub Pages.

## Copy-paste deploy commands

Replace `YOUR_REPO_URL` with your GitHub repository URL, then run:

```powershell
git add .
git commit -m "Create Create Aeronautics landing page"
git remote add origin YOUR_REPO_URL
git push -u origin main
```

Or run the helper script:

```powershell
.\publish.ps1 -RepoUrl YOUR_REPO_URL
```

## GitHub Pages setup

1. Open the repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Set `Build and deployment` to deploy from branch.
4. Choose `main` and `/ (root)`.
5. Save.
