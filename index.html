About

A server built around engineering and flight

Create Aeronautics combines the building depth of Create with the skybound mechanics of Aeronautics to create a world where every engine, propeller, and airframe feels purposeful.

This is not a simple creative sandbox. It is a polished multiplayer experience focused on real flight physics, technical builds, and stylish industrial bases.

What makes it different

  • Airships and aircraft with weight, lift, and thrust that feel grounded.
  • Industrial visuals inspired by brass, copper, and heavy machinery.
  • Chill community vibes with no griefing.

Community

Build together

The server is meant for players who enjoy collaboration, experimentation, and a relaxed place to fly, engineer, and explore with friends.

Zero griefing, high creativity, and a focus on good vibes.
How to Join

Three steps to get airborne

1

Download the modpack

Grab the server pack or launcher profile that includes Create and Aeronautics so your client matches the server setup.

2

Install NeoForge 1.21.1

Use Minecraft 1.21.1 with NeoForge, then import the modpack and confirm that all required mods are enabled.

3

Launch and connect

Open the game, join the server, and start building ships, machines, and hangars with the community.
